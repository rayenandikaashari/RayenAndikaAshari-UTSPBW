Pemrograman Web

Pertemuan 1 : HTML Dasar

Memahami konsep dasar HTML, termasuk struktur elemen, tag, dan atribut. Mahasiswa akan belajar bagaimana membuat halaman web sederhana dengan elemen-elemen seperti heading, paragraf, daftar, dan gambar.

Pertemuan 2 : PHP

Mengenal dasar-dasar PHP sebagai bahasa pemrograman server-side. Pembahasan meliputi sintaks dasar, variabel, tipe data, dan cara menampilkan output ke halaman web menggunakan PHP.

Pertemuan 3 : Mengontrol Alur Program

Mempelajari bagaimana mengontrol alur program menggunakan percabangan (if-else, switch-case). Mahasiswa akan memahami bagaimana pengambilan keputusan bekerja dalam PHP untuk menciptakan logika yang lebih dinamis.

Pertemuan 4 : Perulangan

Menjelaskan konsep perulangan dalam PHP, seperti for, while, dan do-while. Mahasiswa akan mempelajari bagaimana mengoptimalkan kode dengan menggunakan perulangan untuk mengeksekusi instruksi berulang kali.

Pertemuan 5 : Array

Memahami konsep array dalam PHP, termasuk array numerik dan asosiatif. Mahasiswa akan belajar bagaimana menyimpan dan mengakses banyak data dalam satu variabel, serta cara mengelola data dalam array menggunakan berbagai fungsi bawaan PHP.
